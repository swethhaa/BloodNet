 Blood Donation Management System

 Project Description
The Blood Donation Management System is a web-based application developed to manage blood donors, blood stock, hospitals, and blood requests efficiently.

This system helps in maintaining donor records, tracking available blood units, and processing hospital blood requests. When a hospital requests blood, the system automatically checks availability and approves or rejects the request based on stock.

 Features
- Add, view, edit, and delete donor details
- Manage blood stock inventory
- Store hospital information
- Process blood requests with automatic approval/rejection
- Simple and user-friendly interface

 How to Run
1. Install Python and MySQL.
2. Set up MySQL database 'bloodbank' with tables: donor, bloodstock, hospital, bloodrequest, donation, bloodexpiry.
3. Install dependencies: pip install -r requirements.txt
4. Run the Flask app: python app.py
5. Open http://127.0.0.1:5000 in browser.
6. Login as admin with username: admin, password: admin123
